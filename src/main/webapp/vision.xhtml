Visión

Realizar estadisticas actualizadas mundialmente en base a artículos científicos del Covid-19, desarrollando una gestión efectiva manteniendo al mundo informado. Mediante el control de la página web como sector estratégico, que contribuya una infomacion util y precisa.

Valores

1. Efectividad: Todas las actividades y trabajos realizados en KnowCovid deberán alcanzar los resultados esperados con calidad para así concretar, de manera eficiente y eficaz, los objetivos y metas trazadas.



2. Respeto: Se mantendrá el reconocimiento y consideración a cada persona como ser único, con habilidades especificas y conocimientos amplios.



3. Responsabilidad: Se buscará el cumplimiento oportuno de las tareas encomendadas a cada participante, de manera óptima, mediante la toma consciente de decisiones.



4. Transparencia: Toda acción y comportamiento se llevarán a cabo de forma clara, precisa y veraz.
